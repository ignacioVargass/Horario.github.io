102 Etiquetas HTML

Listado de etiquetas HTML con descripción y ejemplos de uso

Nombre de la Etiqueta Descripción Ejemplo de Uso Elemento Adicional
<a> Define un hipervínculo. <a href="https://www.example.com">Visita Ejemplo</a> Usa target="_blank" para abrir en otra pestaña.
<p> Define un párrafo de texto. <p>Este es un párrafo.</p> Puede contener elementos como <a>, <strong>, <em>.
<img> Incrusta una imagen. <img src="imagen.jpg" alt="Descripción de la imagen"> Usa el atributo alt para accesibilidad.
<div> Define un contenedor. <div>Contenido</div> Para agrupar otros elementos.
<h1> - <h6> Define encabezados de diferentes niveles. <h1>Título</h1> Estructura el contenido.
<ul> Lista no ordenada. <ul><li>Elemento</li></ul> Usa <li> para los elementos.
<ol> Lista ordenada. <ol><li>Elemento</li></ol> Usa <li> en listas numeradas.
<li> Define un elemento en una lista. <li>Elemento de lista</li> Dentro de <ul> o <ol>.
<table> Define una tabla. <table><tr><td>Celda</td></tr></table> Contiene filas y celdas.
<tr> Define una fila en una tabla. <tr><td>Celda</td></tr> Usado dentro de <table>.
<td> Define una celda en una tabla. <td>Celda</td> Dentro de <tr>.
<th> Define un encabezado de columna. <th>Encabezado</th> En la primera fila de la tabla.
<form> Define un formulario. <form><input></form> Contiene campos de entrada.
<input> Campo de entrada en un formulario. <input type="text"> Varios tipos como "text", "password", etc.
<button> Define un botón. <button>Enviar</button> Puede usarse en formularios.
<textarea> Área de texto multilínea. <textarea>Texto</textarea> Usado en formularios.
<select> Menú desplegable. <select><option>Opción</option></select> Usado en formularios.
<option> Define una opción dentro de un select. <option>Opción</option> Dentro de <select>.
<label> Etiqueta para un campo de formulario. <label for="nombre">Nombre</label> Asociado a un <input>.
<fieldset> Agrupa elementos en un formulario. <fieldset><legend>Grupo</legend></fieldset> Usado para organizar formularios.
<legend> Título para un <fieldset>. <legend>Título</legend> Dentro de <fieldset>.
<article> Define contenido independiente. <article>Artículo</article> Sección independiente en una página.
<aside> Contenido relacionado lateral. <aside>Contenido lateral</aside> Usado para barras laterales.
<footer> Pie de página. <footer>Pie</footer> Normalmente al final de la página.
<header> Encabezado de sección. <header>Encabezado</header> Usado en la parte superior de una página.
<nav> Define una sección de navegación. <nav>Enlaces</nav> Para menús de navegación.
<section> Define una sección genérica. <section>Sección</section> Usado para agrupar contenido.
<main> Contenido principal de un documento. <main>Contenido principal</main> Usado una vez por página.
<details> Define contenido desplegable. <details><summary>Resumen</summary>Texto</details> Contenido oculto expandible.
<summary> Título para un <details>. <summary>Título</summary> Encabezado para <details>.
<audio> Incrusta audio. <audio controls><source src="audio.mp3"></audio> Incluye controles de reproducción.
<video> Incrusta video. <video controls><source src="video.mp4"></video> Incluye controles de reproducción.
<source> Fuente para <audio> o <video>. <source src="video.mp4" type="video/mp4"> Usado dentro de multimedia.
<iframe> Incrusta otro documento. <iframe src="https://www.example.com"></iframe> Usado para incluir contenido externo.
<canvas> Área de dibujo gráfico. <canvas id="miCanvas"></canvas> Usado con JavaScript para gráficos.
<progress> Barra de progreso. <progress value="70" max="100">70%</progress> Muestra el progreso de una tarea.
<meter> Define un valor escalar. <meter value="0.6">60%</meter> Para medidas dentro de un rango conocido.
<blockquote> Cita en bloque. <blockquote>Cita larga</blockquote> Para citas largas.
<q> Cita en línea. <q>Cita corta</q> Para citas cortas en línea.
<code> Define código fuente. <code>console.log("Hola")</code> Usado para mostrar fragmentos de código.
<pre> Texto preformateado. <pre>Texto con espacios</pre> Mantiene formato y espacios en blanco.
<time> Define una hora o fecha. <time datetime="2024-09-09">9 de Septiembre 2024</time> Para representar tiempos y fechas.
<mark> Resalta texto. <mark>Texto resaltado</mark> Usado para destacar texto.
<abbr> Abreviatura. <abbr title="HyperText Markup Language">HTML</abbr> Define una abreviatura con su significado.
<del> Texto eliminado. <del>Texto borrado</del> Texto tachado, útil para revisiones.
<ins> Texto insertado. <ins>Texto agregado</ins> Texto subrayado, útil para revisiones.
<sup> Texto en superíndice. <sup>2</sup> Útil para potencias o referencias.
<sub> Texto en subíndice. <sub>H</sub>2O Útil para fórmulas químicas.
<hr> Línea horizontal. <hr> Usado para separar contenido.
<br> Salto de línea. <br> Usado para saltar a la siguiente línea.
<wbr> Posible salto de línea. <wbr> Permite saltar línea si es necesario.
<bdi> Texto independiente de la dirección. <bdi>Texto independiente</bdi> Usado para aislar texto en lenguajes bidireccionales.
<bdo> Establece la dirección del texto. <bdo dir="rtl">Texto de derecha a izquierda</bdo> Usado para cambiar la dirección del texto.
<svg> Define gráficos vectoriales escalables. <svg><circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" /></svg> Se usa para gráficos como círculos, rectángulos, etc.
<path> Define una ruta en un gráfico SVG. <path d="M150 0 L75 200 L225 200 Z" /> Se utiliza para crear formas complejas.
<circle> Define un círculo en SVG. <circle cx="50" cy="50" r="40" fill="blue" /> Útil en gráficos vectoriales.
<rect> Define un rectángulo en SVG. <rect width="100" height="100" fill="green" /> Utilizado para dibujar rectángulos.
<g> Agrupa elementos en SVG. <g><rect width="50" height="50" /><circle cx="100" cy="100" r="40" /></g> Agrupa elementos gráficos.
<figcaption> Define un pie de imagen en <figure>. <figcaption>Texto explicativo</figcaption> Da contexto a la imagen o gráfico.
<figure> Define un contenedor para contenido gráfico o multimedia. <figure><img src="image.jpg" /><figcaption>Descripción de la imagen</figcaption></figure> Agrupa imágenes con su pie de foto.
<map> Define un mapa de imagen. <map name="mapa"><area shape="rect" coords="34,44,270,350" href="https://example.com" /></map> Se utiliza con imágenes para áreas clicables.
<area> Define áreas dentro de un mapa de imagen. <area shape="rect" coords="34,44,270,350" href="https://example.com" /> Define zonas clicables en una imagen.
<object> Embebe contenido externo. <object data="movie.swf" type="application/x-shockwave-flash"></object> Permite incluir multimedia o documentos externos.
<embed> Embebe contenido externo. <embed src="video.mp4" width="400" height="300"> Similar a <object>, pero más sencillo.
<track> Define pistas de texto para <audio> o <video>. <track kind="subtitles" src="subtítulos.vtt" srclang="es" label="Español"> Agrega subtítulos o descripciones de audio.
<datalist> Proporciona una lista de opciones predefinidas para un <input>. <datalist id="colores"><option value="Rojo"><option value="Verde"></datalist> Se usa con campos de entrada para sugerencias automáticas.
<keygen> Genera un par de claves (obsoleto en HTML5). <keygen name="name"> Genera claves públicas y privadas.
<output> Muestra el resultado de un cálculo. <output name="result">Resultado</output> Usado en formularios para mostrar resultados.
<template> Define plantillas de contenido reutilizable. <template><p>Texto dentro de plantilla</p></template> No se muestra en la página hasta que es activado con JavaScript.
<picture> Define un contenedor para imágenes con diferentes resoluciones. <picture><source media="(min-width: 650px)" srcset="img_large.jpg"><img src="img.jpg" alt="Descripción de la imagen"></picture> Se usa para imágenes responsivas.
<noscript> Contenido alternativo para navegadores sin JavaScript. <noscript>JavaScript no está habilitado.</noscript> Usado cuando JavaScript está deshabilitado.
<optgroup> Agrupa opciones en un <select>. <optgroup label="Frutas"><option>Manzana</option></optgroup> Agrupa opciones relacionadas.
<fieldset> Agrupa controles relacionados en un formulario. <fieldset><legend>Información personal</legend><input type="text" name="nombre"></fieldset> Proporciona estructura y organización a los formularios.
<legend> Proporciona un título para el grupo <fieldset>. <legend>Título del grupo</legend> Sirve como título o descripción para un conjunto de controles de formulario.
<summary> Define un título visible para el elemento <details>. <summary>Detalles adicionales</summary><details>Aquí están los detalles.</details> Se usa para títulos de bloques colapsables.
<details> Define detalles adicionales que el usuario puede ver u ocultar. <details><summary>Detalles adicionales</summary><p>Texto oculto</p></details> Útil para bloques de contenido expandibles.
